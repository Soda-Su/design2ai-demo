# Example: Architecture Background

User request:

> I studied architecture and want to build an AI portfolio project in six weeks. I know Figma and a little HTML/CSS, but I do not know React. I want something that feels relevant to spatial design.

Expected coaching direction:

- Recommend a critique or workflow demo, not a generic chatbot.
- Example concept: "Precedent-to-design-constraint assistant" that helps students turn a project brief and 20 precedent notes into early design constraints.
- Use JSON/CSV for precedent data first.
- Avoid RAG until the learner has enough source material and a working search problem.
- Week 1 should narrow the design workflow.
- Week 2 should build a single-page shell.
- Week 3 should connect Claude API through a server route.
- Week 4 should structure output as constraints, risks, and next design moves.
- Week 5 should deploy.
- Week 6 should test with messy briefs and weak precedent notes.

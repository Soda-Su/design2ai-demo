# Example: UX Background

User request:

> I am a UX designer. I have interview notes from a class project and want to turn them into an AI demo for my portfolio. I can use basic JavaScript but have never deployed an app.

Expected coaching direction:

- Recommend a workflow demo: "Interview-note insight clustering assistant."
- Use 15-30 anonymized notes as JSON/CSV.
- Avoid database and auth for the first version.
- Emphasize review/edit steps so the AI does not look like a black box.
- Structured output should include theme, supporting quote, confidence, and design implication.
- Week 6 testing should include ambiguous notes and contradictory notes.
